# 一、学术圈的头衔
1、院士：包括中国科学院院士与中国工程院院士，科学院、工程院评选，逢奇数年评选一次。是一个称号。
2.1、国家杰出青年科学基金获得者：简称杰青，国家自然科学基金委评，每年一次，<45岁为硬性条件，是一个基金。
2.2、长江学者：教育部评选，每年一次，<55岁为硬性条件，是一个称号。
2.3、“万人计划”领军人才：纯本土派的顶配，包括杰出人才、领军人才、青年拔尖人才，中组部评选，每年一次。
2.4、海外高层次人才引进计划（千人计划）：现在一般成为国家级高层次人才（海外），专门引进海外正教授回国，中组部评选，不定期。
3.1、国家自然科学基金优秀青年科学基金项目·海外（青千）：海外回国的博士、博士后。国家自然科学基金委评选，每年一次，<40岁为硬性条件。
3.2、国家优秀青年科学基金获得者（优青）：俗称小杰青。国家自然科学基金委评选，每年一次，男<38，女<40为硬性条件。
3.3、万人计划青年拔尖人才（青拔）：中组部评选，每年一次。
3.4、青年长江学者：教育部评选，每年一次。

# 二、资金来源
理工：国自然（面上、青年）、科协青年人才托举工程、科技部国家重点研发、横向
文科、社科：国家社会科学基金委。
